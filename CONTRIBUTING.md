# Guía de Contribución 🤝

¡Gracias por tu interés en mejorar **FLL-SYSTEM**! Sigue estos pasos para contribuir:

## 📜 Reglas de Oro
- Mantén el código limpio y legible.
- **Toda nueva funcionalidad debe incluir tests.**
- Sigue las convenciones de nombres del proyecto.

## 🛠️ Proceso de Desarrollo
1. Haz un **Fork** del repositorio.
2. Crea una rama para tu feature: `git checkout -b feature/nueva-mejora`.
3. Realiza tus cambios y asegúrate de que todos los tests pasen: `npm test`.
4. Envía un **Pull Request** detallando tus cambios.

## 🧪 Estándares de Testing
- Pruebas unitarias en `*.test.js` o `*.test.tsx`.
- Mockea dependencias externas (bases de datos, sockets).
- Usa `supertest` para pruebas de nuevos endpoints.

---
Tu ayuda es fundamental para que FIRST LEGO League sea una experiencia increíble para todos. 🤖✨
